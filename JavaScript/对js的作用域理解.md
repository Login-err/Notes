1. 全局作用域

    顶层环境中申明的变量都是全局作用域，他们的属性其实都是在window下面

2. 局部作用域

    es6新增的特性，弥补了以前只能使用匿名及时运行函数来创建局部变量的缺陷，用let和const申明即可

3. 作用域链

    我个人认为和原型链的理解差不多，

    ```javascript
    function a(){
        const a = [];
        function b(){
            console.log(a)
        }
    }
    // 在b函数的作用域内，寻找a变量，没有找到，再向上层作用域寻找a变量。因此形成了一条作用域链
    ```
